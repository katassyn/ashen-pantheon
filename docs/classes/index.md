# Classes

Ashen Pantheon has **three base classes**. Each class shapes your early playstyle and determines which ascendancies you can choose at level 50. The class you pick at character creation is permanent.

| Class | Role | Ascendancies |
|---|---|---|
| [Warrior](warrior.md) | Melee frontline, high HP, physical power | Berserker · Guardian · Paladin |
| [Mage](mage.md) | Ranged caster, elemental mastery, summons | Elementalist · Arcanist · Hexblade |
| [Archer](archer.md) | Ranged precision, mobility, pet synergies | Shadowstalker · Beastmaster · Pathfinder |

## Choosing a Class

Your base class affects:

- **Starting stats** — health, mana, stamina distribution
- **Skill tree shape** — each class has unique skill tree nodes alongside shared nodes
- **Available ascendancies** — you can only pick from your class's three ascendancy paths
- **Weapon proficiency** — Warriors favor melee weapons; Mages favor staves and orbs; Archers favor bows and daggers
