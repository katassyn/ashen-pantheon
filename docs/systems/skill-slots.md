# Skill Slots

Ashen Pantheon uses a **free keybind system** — any skill you have unlocked can be assigned to any keybind slot, with no restrictions.

## How It Works

1. **Unlock skills** via the skill tree (Skill Unlock nodes) and ascendancy tree
2. Open the **Skills panel** to see all unlocked skills
3. **Drag any skill** to any empty keybind slot
4. Your keybind is now active in combat

## No Restrictions

There is no concept of a "skill type" that limits assignment. You can:

- Fill every slot with damage skills
- Mix damage, defense, and movement skills freely
- Assign healing skills alongside crowd-control skills
- Run zero defensive skills if you prefer maximum aggression (not recommended)

This means the optimal skill layout is a meaningful build decision, not a default.

## Build Implications

The free keybind system interacts with build design in important ways:

- Skill modifier nodes in the skill tree only apply when the modified skill is slotted
- Some ascendancy nodes grant passive bonuses to "any movement skill slotted"
- You can create highly specialized setups (all damage) or flexible setups (mixed toolkit)
