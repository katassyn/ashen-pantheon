# Ascendancy Tree (Levels 50–100)

The ascendancy tree is the second phase of character progression, unlocked after you choose your ascendancy at level 50. It is a smaller, more powerful tree exclusive to your chosen ascendancy path.

## Overview

- **25 points** available (one per 2 levels from 52 to 100)
- Tree is **exclusive to your ascendancy** — Berserker and Guardian trees are completely different
- Nodes are fewer but far more impactful than main skill tree nodes
- Tree is designed to amplify and complete the build direction started in the main tree

## Pacing

| Level Range | Points Earned | Cumulative Total |
|---|---|---|
| 50 | 0 (tree unlocked, no points yet) | 0 |
| 52 | 1 | 1 |
| 54 | 1 | 2 |
| ... | ... | ... |
| 100 | 1 | 25 |

## Node Power Level

Ascendancy nodes are intentionally strong. Examples of the tier of power:

- "Your critical strikes deal 40% more damage"
- "Lifesteal heals for double the amount when below 50% health"
- "Taunt now pulls all enemies in the instance to your location"

These are build-defining, not incremental.

## Synergy with Main Tree

The ascendancy tree assumes you have built the main tree to support it. For example, Berserker ascendancy nodes that scale with Rage assume you have unlocked Rage-generating skills in the main tree.

Plan your main tree with your intended ascendancy in mind from character creation.
