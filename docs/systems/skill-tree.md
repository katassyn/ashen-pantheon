# Skill Tree (Levels 1–50)

The main skill tree is the foundation of every build in Ashen Pantheon. It is a large, branching network of nodes that you navigate from level 1 through level 49, spending one point per level.

## Overview

- **49 points** available (one per level, levels 1–49)
- Tree is **shared across classes** at the outer edges, with **class-exclusive sections** in the center
- Node types: passive stats, active skill unlocks, and skill modifiers
- You cannot fill the entire tree — builds are defined by what you choose to skip

## Node Types

| Type | Description |
|---|---|
| **Passive Stat Node** | Directly increases a stat (e.g., +5% physical damage, +20 max health) |
| **Skill Unlock Node** | Adds a new active skill to your available skill pool |
| **Modifier Node** | Changes the behavior of a specific skill (e.g., "Fire Ball now hits twice but costs 20% more mana") |
| **Keystone Node** | Rare, powerful nodes that fundamentally change how a mechanic works. Located deep in the tree. |

## Class Sections

Each class has an exclusive section of the tree that only they can access:

| Class | Tree Section Focus |
|---|---|
| Warrior | Melee damage, armor scaling, stance nodes |
| Mage | Elemental affinity, cast speed, mana efficiency |
| Archer | Range multipliers, mobility skills, pet slot nodes |

## Build Philosophy

The skill tree rewards specialization. Spreading points evenly produces a weak generalist build. Strong builds commit to a clear path:

1. Rush toward key Skill Unlock nodes early
2. Fill in passive stat nodes along the path
3. Pick up Modifier nodes for your core active skills
4. Consider Keystone nodes as long-term goals
