# Game Overview

Ashen Pantheon is a top-down 2D instanced action RPG.

## Core Specs

| Feature | Detail |
|---|---|
| Genre | Top-down 2D Action RPG |
| Level Cap | 100 |
| Base Classes | 3 — Warrior, Mage, Archer |
| Ascendancies | 9 — chosen permanently at level 50 |
| Max Party Size | 4 |
| World | Player-owned instances + global layer (chat, rankings, trading) |
| Boss Mythologies | Greek, Norse, Egyptian, Japanese, Slavic |

## Progression

- **Levels 1–49:** main skill tree (49 points).
- **Level 50:** choose one of three ascendancies for your class (permanent).
- **Levels 51–100:** ascendancy tree (25 points — 1 per 2 levels).

## Bosses

Multi-phase encounters drawn from Greek, Norse, Egyptian, Japanese, and Slavic mythologies.
