# Game Overview

Ashen Pantheon is set in a fractured realm where the boundaries between mythological pantheons have collapsed. Ancient gods — stripped of their divine order — have gone mad, their power seeping into a world of ash and ruin. You are an **Ashen Hunter**: a mortal who has absorbed enough divine essence to stand against them.

## Core Concept

The game blends responsive top-down action combat with the depth of an ARPG progression system. Every encounter is hand-crafted, every boss tells a story, and every build decision carries weight. The world is yours — but the gods want it back.

## The Instanced World

Each player owns their own **world instance** — a personal version of the game map that only they control. You explore, clear zones, and progress at your own pace without interference from other players.

When you want company, you can invite up to **3 friends** to join your instance, forming a party of 4.

### Global Layer

Even in an instanced world, you're never truly alone. A persistent **global layer** connects all players across instances:

| Layer Feature | Description |
|---|---|
| Chat | Cross-instance channels — general, trade, guild |
| Rankings | Leaderboards for level, boss kills, gear score |
| Trading | Player-to-player item exchange across instances |

## Progression at a Glance

| Phase | Levels | What Changes |
|---|---|---|
| Early Game | 1 – 49 | Main skill tree — broad, many nodes, class skills unlocked |
| Ascendancy | 50 | Choose one ascendancy path (permanent — choose wisely) |
| Mid/Late Game | 51 – 100 | Ascendancy tree — 1 point per 2 levels = 25 total points, fewer but far stronger nodes |
| Endgame | 100 | Gear farming, challenge bosses, rankings |

## The Bosses

Every major boss in Ashen Pantheon is drawn from real-world mythology. Each boss has a unique visual identity, lore connection, and a **multi-phase fight** that escalates in scope and intensity.

**Mythologies represented:**

- **Greek** — Olympian gods and ancient monsters
- **Norse** — Aesir, Vanir, and primordial beasts
- **Egyptian** — Ennead deities and pharaonic guardians
- **Japanese** — Kami and yokai of the spirit world
- **Slavic** — Ancient gods and forest spirits
