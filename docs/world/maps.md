# Maps & Zones

The Ashen Realm is divided into **zones** — distinct geographic regions that players explore and clear. Each zone has a distinct visual theme drawn from one of the game's mythological traditions.

## Zone Structure

Each zone contains:

| Element | Description |
|---|---|
| **Exploration Area** | Open map space with enemy encounters and environmental storytelling |
| **Resource Nodes** | Gatherable materials for crafting and trading |
| **Hidden Areas** | Optional side paths with lore items and rare enemies |
| **Boss Arena** | A locked area that opens when triggered — contains the zone's boss |
| **Instance Checkpoint** | Saves your position in the instance so you re-enter in the same zone |

## Mythological Zone Themes
